このミッションでは 12x9 (A1 から L9) の六角形の升目(ヘクス)を使います。

  • その仕様は:
    • 平面が上
    • アルファベットが列、数字が行を表す
    • 偶数の列が下に突出

入力値として月面の障害物ヘクスの集合(set)が与えられます。 着陸地点の候補となる全てのヘクスを集合(set)として返して下さい。

  • 着陸候補ヘクスの条件:
    • 障害物ヘクスでないこと
    • 等距離 (1以上) のヘクスに障害物ヘクスがないこと
    • 上記の半径が最長であること
注意:
  • 12x9 の升目の外側は常に障害物ヘクスと想定します
  • もし着陸候補ヘクスが無ければ空の集合(set())を返すこと

例: 

assert landing_site({'E5', 'E7', 'F4', 'F6', 'G4', 'G6', 'H3', 'H5'}) == {'C3', 'J7'}
assert landing_site({'A4', 'C2', 'C6', 'C9', 'D4', 'D7', 'F1', 'F5',
                     'F8', 'G4', 'H7', 'I2', 'I5', 'I9', 'K3', 'K8', 'L5'}) == {'B7', 'E3', 'J6'}

入力: 月面の障害物ヘクス (文字列の集合)

出力: 全ての着陸候補ヘクス (文字列の集合)

前提条件:

  • all(re.fullmatch('[A-L][1-9]', i) for i in input)
